# TEMPERATURE CONVERTER PROGRAM
A simple program for converting temperature to a different form
---

## FEATURES OF THE PROGRAM
this program can convert all type of temperature, such as:
- Celcius
- Fahrenheit
- Kelvin
- Rankie

## CLASSES
there are 3 classes:
- Main
- Temperature
- TemperatureConverter
